## <a name="introduction">🤖 Introduction</a>

**CoreDrive** is a storage management and file sharing platform that allows users to upload, organize, manage, and share files in a clean and intuitive way.

I built this project to practice building a full-stack product with modern tools and to create something that feels both practical and polished. The platform focuses on seamless file management, responsive design, and a user-friendly experience across devices.

The name **CoreDrive - The heart of your data** reflects the goal of the project: creating a central place where users can store, access, and manage what matters most to them.

## <a name="features">🔋 Features</a>

👉 **User Authentication**  
Secure signup, login, and logout functionality powered by Appwrite authentication.

👉 **File Uploads**  
Users can upload different file types such as documents, images, videos, and audio files.

👉 **View and Manage Files**  
Files can be viewed, organized, renamed, and deleted through a simple and modern interface.

👉 **File Downloading**  
Users can quickly download their files whenever needed.

👉 **File Sharing**  
Files can be shared with other users, making collaboration easier and more convenient.

👉 **Dashboard Overview**  
A dashboard provides a clear summary of total storage, used storage, recent uploads, and file categories.

👉 **Global Search**  
Users can search for files and shared content across the platform.

👉 **Sorting Options**  
Files can be sorted by name, date, size, and other criteria to improve organization.

👉 **Responsive Design**  
The interface is designed to work smoothly on desktop, tablet, and mobile devices.

👉 **Reusable Code Structure**  
The project is built with reusable components and a scalable structure for easier maintenance and future improvements.
